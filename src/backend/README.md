# Backend

Esta carpeta contendrá toda la lógica de negocio y servicios del lado del servidor de la aplicación Contigo.

## Estructura

- **api/**: Endpoints y controladores de API
- **models/**: Modelos de datos y esquemas de base de datos
- **services/**: Lógica de negocio y servicios de aplicación
- **utils/**: Utilidades de backend (validación, autenticación, helpers, etc.)

## Propósito

Esta carpeta agrupa todo el código relacionado con la lógica de negocio, acceso a datos y servicios del servidor, manteniendo una clara separación de responsabilidades con el frontend.
