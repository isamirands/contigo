# Frontend

Esta carpeta contiene todos los componentes relacionados con la interfaz de usuario de la aplicación Contigo.

## Estructura

- **components/**: Componentes React reutilizables (botones, modales, tarjetas, etc.)
- **pages/**: Páginas completas de la aplicación (Home, Journey, Settings, etc.)
- **styles/**: Archivos de estilos CSS globales y específicos de componentes
- **utils/**: Utilidades de frontend incluyendo hooks personalizados y funciones auxiliares

## Propósito

Esta carpeta agrupa todo el código relacionado con la presentación y la interacción del usuario, manteniendo una clara separación entre la lógica de frontend y backend.
