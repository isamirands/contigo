# Shared

Esta carpeta contiene código compartido entre frontend y backend, promoviendo la reutilización y consistencia.

## Estructura

- **types/**: Tipos TypeScript compartidos (interfaces, types, enums)
- **constants/**: Constantes y configuraciones compartidas (datos estáticos, configuraciones, etc.)

## Propósito

Esta carpeta agrupa todo el código que es utilizado tanto por el frontend como por el backend, evitando duplicación y asegurando consistencia en tipos y constantes a través de toda la aplicación.
