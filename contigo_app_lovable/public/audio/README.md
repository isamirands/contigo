# Audio Files

## Archivo Requerido

Sube tu archivo de audio aquí con el nombre:

**`hipoglucemia-hiperglucemia.wav`**

## Formato Recomendado

- **Formato**: WAV o MP3
- **Nombre**: `hipoglucemia-hiperglucemia.wav` (o `.mp3`)
- **Duración**: ~240 segundos (4 minutos)
- **Sincronización con slides**:
  - Slide 1: 0-59 segundos
  - Slide 2: 60-119 segundos
  - Slide 3: 120-179 segundos
  - Slide 4: 180-240 segundos

## Después de Subir el Archivo

El código ya está preparado para usar el audio automáticamente.
La ruta será: `/audio/hipoglucemia-hiperglucemia.wav`

## Nota

Si usas MP3 en lugar de WAV, cambia la extensión en el código de `EducationalModal.tsx`
